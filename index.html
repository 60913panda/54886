系統設定指南

一、Gemini API (文字辨識)

  1. 前往 Google AI Studio。
  2. 點選 Get API key > Create API key。
  3. 複製金鑰貼回本工具。

二、Google Sheets API (自動寫入)

API 模式專用，簡易模式無需設定。

  1. 前往 Google Cloud Console 建立專案。
  2. 啟用 Google Sheets API。
  3. 設定 OAuth 同意畫面 (外部, 加入測試者 Email)。
  4. 建立憑證：
    • API Key (複製貼上)。
    • OAuth Client ID (網頁應用程式，來源加入 localhost)。
