API 金鑰設定教學

一、Gemini API (文字辨識用 - 必備)

  1. 前往 Google AI Studio 並登入。
  2. 點選左上角的 Get API key 按鈕。
  3. 點選 Create API key (Create key in new project)。
  4. 複製產生的 API Key，貼回本工具左側的「Gemini API 金鑰」欄位並儲存。

二、Google Sheets API (自動寫入用 - 選用)

僅在使用「API 模式」直接寫入 Google 試算表時需要此設定。若使用簡易模式可跳過。

  1. 1. 建立專案： 前往 Google Cloud Console，建立一個新專案。
  2. 2. 啟用 Google Sheets API： 在左側選單點選「API 和服務」 > 「資料庫」，搜尋 Google Sheets API 並啟用它。
  3. 3. 設定 OAuth 同意畫面：
    • 點選「API 和服務」 > 「OAuth 同意畫面」。
    • User Type 選擇「外部 (External)」。
    • 填寫應用程式名稱和聯絡信箱。
    • 在「範圍 (Scopes)」步驟，搜尋並新增 .../auth/spreadsheets。
    • 在「測試使用者」步驟，加入您自己的 Gmail 帳號。
  4. 4. 建立憑證 (Credentials)：

    點選「API 和服務」 > 「憑證」，點擊上方「建立憑證」。

    • API 金鑰： 建立後複製下來。這是您的 Google Cloud API Key。
    • OAuth 用戶端 ID：
      選擇應用程式類型為「網頁應用程式」。
      在「已授權的 JavaScript 來源」中，如果您是在本機執行，請加入 http://localhost 和 http://127.0.0.1。如果是透過 file:// 開啟，可能需要架設簡易的本地伺服器。
      建立後，複製您的「用戶端編號 (Client ID)」。這是您的 Google Cloud Client ID。
